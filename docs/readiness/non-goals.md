# Non-Goals

The implementation will not add capability beyond the approved plan:

- no hidden feature parity with reference systems
- no offline card payments
- no offline gift card or membership mutations
- no payroll filing, withholding, ACH, or compliance automation
- no delivery payout reconciliation
- no KDS scope beyond printer-oriented workflows defined in the PRD
- no migration/import adapters from other POS products in this PRD
